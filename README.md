# 🟢 A2G Carousel Generator

A powerful, fully offline **social media carousel builder** that runs entirely 
in your browser. Design beautiful multi-slide carousels for Instagram, LinkedIn, 
Twitter/X, and more — then export them as PNG, JPG, or WebP at any resolution.

No React. No Node. No server. Just open `index.html` and start designing.

---

## ✨ Features

### 🎨 Design
- Live canvas editor (540×540 design viewport)
- Dark theme with customizable **accent color** and **background color**
- 6 background patterns (Dots, Grid, Lines, Crosses, Diagonal, None)
- Custom background image upload (cover fit)
- Brand color swatches palette (save and reuse your brand colors)
- Logo upload with custom width/height controls

### 🗂 Slides
- 4 slide templates: **Intro**, **Step**, **CTA**, **Blank**
- Pre-populated default elements per template
- Drag-to-reorder slide cards
- Top tab bar for fast slide switching
- Unlimited slides per project

### 🧩 Elements (60+)
Organized in 8 categories with **live search**:

| Category | Elements |
|---|---|
| **Text** | Heading, Sub-heading, Body, Caption, Quote, Bullet, Stat, Hashtag, Handle, Testimonial, Tip, and more |
| **Media** | Image, Profile Photo, Wide Photo, Rounded Image, Icon/Badge, Logo |
| **Shapes** | Rectangle, Rounded Rect, Card BG, Accent Bar, Tag Pill, Gradient Rect, Circle, Glow Circle, Diamond |
| **Lines** | Line, Divider, Dashed Line, Short Line, Vertical Line |
| **UI** | Button, Ghost Button, Arrow Button, Progress Dots, Nav Arrow, Frame Border |
| **Steps** | Step Number, Step Label, Check Item, Date Label |
| **Stats** | Big Stat, Stat Label, VS Badge, Star Rating, Percentage, Growth Arrow |
| **Brand** | Brand Name, Tagline, Website URL, CTA Text, Offer Badge, Swipe Prompt |

### ✏️ Element Controls
- **Drag** to move elements anywhere on canvas
- **Resize** with 8-point handles (corners + edges)
- **Property panel** — edit text, font, size, color, opacity, shadow, stroke, radius, alignment, and more
- Per-element **visibility toggle** (👁) and **lock toggle** (🔒)
- **Duplicate**, **delete**, **right-click context menu**
- Snap-to-grid alignment
- Keyboard shortcuts (arrow keys, Del, Ctrl+C/V/D/Z/Y/S)
- Multi-select support

### 📤 Export
- Export **current slide** or **all slides at once**
- Format: **PNG** (lossless), **JPG**, **WebP**
- Quality slider for JPG/WebP
- Built-in size presets:
  - Instagram Post (1080×1080)
  - Instagram Story (1080×1920)
  - IG Landscape (1080×566)
  - Full HD (1920×1080)
  - Twitter/X (1200×675)
  - LinkedIn (1200×627)
  - Custom dimensions
- Exports via Canvas API — pixel-perfect output

### 💾 Save / Load
- Save full project to a `.json` file
- Load project from file to continue editing
- Undo / Redo history (up to 50 steps)
